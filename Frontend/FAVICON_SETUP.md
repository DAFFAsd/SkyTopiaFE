## ğŸ”„ How It Works

1. **Next.js Convention**: File named `icon.tsx` in `app/` directory automatically becomes favicon
2. **Dynamic Generation**: Uses `ImageResponse` to generate PNG on-the-fly
3. **Caching**: Browser caches the generated icon
4. **Fallback**: Multiple icon sources in metadata for compatibility

## ğŸ“ Customization Options

### Change Icon Size
```typescript
export const size = {
  width: 48,  // Change to 48x48 for larger icon
  height: 48,
};
```

### Use Different Design
Replace the JSX in `Icon()` function with your custom design:
```typescript
<div style={{ /* your custom styles */ }}>
  {/* your custom content */}
</div>
```

### Use SVG Directly
If you prefer to use the SVG file directly without conversion:
1. Rename `public/skytopia-icon.svg` to `app/icon.svg`
2. Delete `app/icon.tsx`
3. Next.js will use the SVG automatically
